Integrate GNews With Blink

Learn how to start using Blink by adding in a GNews API key.

  1. Go to GNews.io and create a free account.
  2. Verify your email address as prompted.
  3. On the dashboard, click the copy icon on the API key section.
  4. Paste the key into Blink when prompted for adding a GNews API key.
  5. You're all set to start reading the news you care about!
