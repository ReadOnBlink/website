Integrate OpenAI With Blink

Learn how to harness the power of AI with Blink through OpenAI.

  1. Go to Openai.com and create a free account.
  2. Follow the signup instructions as prompted.
  3. Click the icon with your account name in the top right corner of the screen and click "Vuew API Keys".
  4. Generate a new API key by clicking the "Create new secret key" button. (IMPORTANT: Make sure this key is kept safe and private!)
  5. Copy the API key and head back over to Blink.
  6. Paste the API key into the text box asking for an optional OpenAI API key.
  7. Click "Continue" and you're all set! You successfully integrated OpenAI with your Blink account!
